
# SpaceX API

Es una prueba del uso de la API de SpaceX en la cual se listan los lanzamientos asi como un detalle de cada uno, ademas de mostrar los cohetes.


## Features

- Listado de lanzamientos
- Detalle de lanzamiento
- Listado de Cohetes
